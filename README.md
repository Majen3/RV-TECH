# RV-TECH
I was given a data set to work on and find some basic analysis questions, like total sales, top 5 products by sales, and the monthly trend. 
Using tools like Excel and Power BI, the first step I took in order to get this analysis was to duplicate the dataset and clean the duplicated one, so as to show the original dataset and the clean one.
The next thing I did was to find duplicates in the dataset, but there was none, so I proceeded to the next stage which was looking for null values, in the process of doing that, I found out that both profits and losses were on the same column, which was named PROFIT, I seperated them using the tool text-to-column, then for the empty cells, I used find and replaced tool to replace it with 0.
I converted some numerical data to currencies.
After I was sure that my dataset was ready, I moved to the analysis process. With the help of the pivot table tool, I was able to summarize and find my analysis on the top 5 products by sales and monthly trend.
To be able to find my total sales, I used the formula "=SUM()".
After the analysis was done, I moved to Power BI to visualize my data, I was able to use the card tool to display my total sales, average sales, and number of sales that took place for the past 4 years.
Using a clustered column chart, I was able to visualize the top 5 products by sales, and using a line chart and a slicer, I was able to visualize the monthly trend in each year.
Lastly, I made a short Power Point Presentation, explaining my findings.
